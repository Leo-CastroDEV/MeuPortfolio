# 💻 Portfólio | Leonardo Castro

Bem-vindo ao meu portfólio!

Este projeto foi desenvolvido para apresentar minha trajetória como estudante de **Análise e Desenvolvimento de Sistemas**, além de demonstrar minhas habilidades em desenvolvimento Front-end utilizando apenas tecnologias nativas da web.

---

## 📖 Sobre

O objetivo deste portfólio é reunir informações sobre mim, minhas competências e projetos desenvolvidos durante minha jornada de aprendizado na programação.

Todo o projeto foi construído com foco em:

- Código limpo
- Organização
- Responsividade
- Design moderno
- Boas práticas de HTML e CSS

---

## 🚀 Tecnologias Utilizadas

- CSS3

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/Leo-CastroDEV/MeuPortfolio.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` em qualquer navegador.

Não é necessário instalar dependências.
